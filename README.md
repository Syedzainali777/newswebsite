# 📰 News Website (MERN Stack)

A full-stack news publishing platform built with the MERN stack, featuring a powerful admin panel for managing articles, categories, and content in real time.

---

## 🚀 What This Project Does

This application allows users to read the latest news while enabling admins to create, edit, and manage blog posts through a dedicated dashboard.

- 🌐 Public news website for users  
- ✍️ Admin panel to publish and manage blogs  
- 📂 Category-based news organization  
- 🔍 Search and filter functionality  
- 📱 Responsive design for all devices  

---

## 📊 Key Impact

- Simplifies content publishing without technical knowledge  
- Enables real-time content updates via admin dashboard  
- Scalable architecture for modern blogging/news platforms  
- Separation of concerns between frontend and backend  

---

## 🛠️ Tech Stack

### Frontend
- React.js  
- Tailwind CSS  

### Backend
- Node.js  
- Express.js  

### Database
- MongoDB  

### Other
- JWT Authentication  
- REST APIs  

---

## ✨ Core Features

### 👤 User Side
- Browse latest news articles  
- View posts by category  
- Search functionality  
- Responsive UI  

### 🔐 Admin Panel
- Secure login (JWT-based authentication)  
- Create, update, delete blog posts  
- Manage categories  
- Upload images (if implemented)

## ⚙️ Installation

```bash
git clone https://github.com/your-username/news-website
cd news-website
````

### Install dependencies

```bash id="installcmd"
# frontend
cd client
npm install

# backend
cd ../server
npm install
```

---

## ▶️ Run the Project

```bash id="runcmd"
# run backend
cd server
npm run dev

# run frontend
cd client
npm start
